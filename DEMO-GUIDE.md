## Quick Setup (5 minutes)

### Step 1: Generate Demo Data
1. Open Chrome and navigate to any webpage
2. Open Chrome DevTools (F12)
3. Go to the Console tab
4. Copy and paste the entire content from `demo-data-generator.js`
5. Press Enter - you'll see confirmation messages
6. Close DevTools

### Step 2: Open the Extension
1. Click the Note-It extension icon in Chrome
2. Click "View Graph" to open the graph visualization
3. You should see a rich graph with 15 interconnected notes!

## Fallback Plans üõü

### If the graph looks empty:
1. Check Chrome DevTools console for errors
2. Re-run the demo data generator script
3. Refresh the extension tab

### If zoom is buggy:
1. Refresh the graph view page
2. Use mouse wheel for zoom, not trackpad gestures
3. Start from a fresh zoom level (Ctrl+0)

### If filtering doesn't work:
1. Clear all filters and start fresh
2. Use exact tag names from the demo data
3. Try simple searches before advanced filtering
